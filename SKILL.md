## Core Frameworks & Mental Models

### 1. The 11 Executive Skills
The foundational taxonomy. Use these to diagnose any child's "scattered" behavior into a specific, actionable deficit. Skills cluster into **Thinking** (Working Memory, Planning/Prioritization, Organization, Time Management, Metacognition) and **Doing** (Response Inhibition, Emotional Control, Sustained Attention, Task Initiation, Goal-Directed Persistence, Flexibility).

| Skill | Definition | Use When… |
|-------|-----------|-----------|
| **Response Inhibition** | Think before acting; resist urges | Child talks out of turn, grabs, interrupts |
| **Working Memory** | Hold info in mind while performing tasks | Child forgets instructions, loses track mid-task |
| **Emotional Control** | Manage emotions to achieve goals | Meltdowns over frustration, test anxiety |
| **Sustained Attention** | Maintain focus despite distraction/boredom | Child starts tasks but doesn't finish |
| **Task Initiation** | Begin tasks without procrastination | Child puts off homework, chores, routines |
| **Planning/Prioritization** | Create a roadmap; decide what matters | Child freezes on long-term projects |
| **Organization** | Create/maintain systems for materials | Loses belongings, messy room, missing homework |
| **Time Management** | Estimate, allocate, and respect time | Always late, underestimates task duration |
| **Goal-Directed Persistence** | Follow through despite competing interests | Quits when not immediately good at something |
| **Flexibility** | Revise plans in face of obstacles | Meltdowns over schedule changes |
| **Metacognition** | Self-monitor and self-evaluate | Rushes through work, unaware of social impact |

### 2. The ABC Intervention Model
Three levers for any behavior change. Use all three for maximum effect.
- **A** (Antecedent) → Modify the environment *before* the behavior occurs (ch06)
- **B** (Behavior) → Teach the skill directly (ch07)
- **C** (Consequence) → Use incentives to motivate performance (ch08)

### 3. The Intervention Hierarchy (Game Plan)
Start with the minimum intervention and escalate only as needed:
1. Environmental modification (ch06) — change the physical/social/task environment
2. Verbal scaffolding, games, informal teaching (ch07)
3. Incentives — praise, then formal systems (ch08)
4. Ready-made daily routine plans (ch10)
5. Targeted skill-by-skill interventions (ch11–21)
6. Professional help (ch22)

### 4. The 10 Principles (ch05)
1. **Teach deficient skills directly** — don't expect osmosis
2. **Consider developmental level** — match expectations to age
3. **Move from external to internal** — change the environment before requiring child change
4. **The external includes three levers** — environment, task, interaction style
5. **Use drive for mastery and control** — routines, choices, Grandma's Law
6. **Modify tasks to match effort capacity** — use the 1–10 Effort Scale; design tasks to feel like a 3
7. **Use incentives to augment instruction** — not bribery; they build delayed gratification
8. **Provide just enough support for success** — too much creates dependence; too little creates failure
9. **Keep supports until mastery** — 3 weeks is optimistic; monitor at monthly intervals
10. **Fade supports gradually, never abruptly** — like bike-riding: hold, release for seconds, catch wobbles

### 5. Goodness of Fit
The match between a task's executive demands and the child's current skills. Poor fit → avoidance, meltdowns, seeming defiance. Also applies to parent-child skill profiles (ch03). Use the **Task-Triage Framework** (ch04) — 7 diagnostic questions — when a child resists a task: assume "can't do" before "won't do," reverse-engineer the task's executive demands, check the environment, and analyze past successes.

### 6. Key Teaching Structures
- **Verbal Scaffolding** (ch07): "Ask, don't tell" — use questions that build thinking rather than commands that build compliance
- **6-Step Direct Teaching Sequence** (ch07): (1) Identify problem behavior specifically → (2) Set goal → (3) Outline steps → (4) Turn into checklist → (5) Supervise → (6) Fade supervision
- **4-Stage Fading** (ch07): Parent directs → visual cues do the thinking → prompting questions → child self-directs
- **Backward Chaining** (ch07, ch10): Start at the last step and work backward — useful for high-effort tasks and cleaning
- **Think-Aloud Procedure** (ch10): Parent models internal decision-making aloud so the child sees how structured thinking works
- **Grandma's Law / Premack Principle** (ch05, ch08): "First have-to, then want-to" — less preferred tasks precede more preferred activities

### 7. Developmental Trajectory
Executive skills emerge from birth through the first two decades. The prefrontal cortex is among the last brain regions to fully mature (late adolescence/early adulthood). Parents serve as the child's "external frontal lobes" — providing planning, organization, and impulse control until the child's brain can take over. **Target ages for this book: 4–14**, because early intervention builds the foundation for high school and adult independence.

### 8. The Effort Scale (1–10)
Ask the child to rate a task's difficulty. Design interventions so the task feels like a 3. Gradually increase effort required. If a task feels like a 10, the child won't start — reduce scope, add structure, or provide immediate reward.

### 9. Parent-as-Coach Model (ch24)
For adolescents: shift from manager to coach. Present information casually and leave it for the child to ponder. When failure comes, resist "I told you so." Provide minimum support necessary for the child to pick themselves up. Use natural and logical consequences rather than arbitrary punishments.
