# Chapter 12: Enhancing Working Memory

## Core Idea
Working memory — the ability to hold information in mind while performing complex tasks — develops from infancy through adolescence, can be systematically supported with visual cues, rehearsal, and environmental structuring, and requires persistent cueing that fades only very gradually.

## Frameworks Introduced

- **Intervention Design Template for Working Memory**: Same 2-step structure as Ch 11.
  - **Step 1: Establish Behavioral Goal** — target executive skill (working memory) + specific behavioral objective with measurable criteria.
  - **Step 2: Design Intervention** — environmental supports (advance preparation, timers, visual aids, parent observation/prompts) + skill instruction (meet to discuss, make list, create visual sequence, walk-through/practice, cue system) + incentives (praise, tangible rewards tied to target).

- **Keys to Success for Working Memory Interventions**:
  - Be enthusiastic and thorough in early stages — novelty + systematic cueing + incentive creates initial success.
  - Err on the side of cueing for too long — when parents withdraw cues too early, gains often disappear; fade in baby steps.
  - Don't rely on the child's verbal assurance that they have acted on your cue — follow up with a visual check.

## Key Concepts
- **Working memory**: The capacity to hold information in mind while performing complex tasks. Examples: remembering what to buy at the store without a list; remembering to stop at the dry cleaner on the way home.
- **Nonverbal vs. verbal working memory**: Nonverbal working memory develops before verbal because it emerges before language; language expands working memory capacity by adding a verbal retrieval channel.
- **Prefrontal cortex workload**: When children and teens perform working-memory tasks, their prefrontal cortex does all the work instead of distributing it to specialized regions (as adults' brains do) — making it more consciously effortful for them.

## Mental Models
- Use "check your list" as the standard prompt — the visual cue does the remembering, not the child's working memory.
- Think of working memory like a whiteboard that gets wiped clean under stress or distraction — written cues are permanent.
- Use the developmental milestones as an expectations checklist — don't expect a 7-year-old to independently remember multistep after-school chores without visual aids.
- Use cell phones/text messages for middle schoolers as a natural working memory prosthetic — they already use these devices constantly.

## Anti-patterns
- **Relying on verbal assurance**: When asked "Did you check?" a child with weak working memory will often say yes and then proceed to forget. Always verify with a visual check.
- **Fading cues too early**: The most common cause of "relapse" in working memory interventions is parents withdrawing visual cues or prompts before the behavior is truly automatic.
- **Giving multistep verbal instructions without visual backup**: "Go upstairs, put your laundry away, get your soccer bag, and meet me at the car" is a working memory test, not a request.
- **Assuming a forgetful child is being willfully disobedient**: Working memory failures look like defiance but aren't — the child genuinely lost the information.
- **Rescuing the child after agreeing not to** (e.g., providing the forgotten shin guards): Undermines the natural consequence that would teach the child to use the system.

## Key Takeaways
1. Make eye contact before telling the child something important; minimize distractions (turn off the TV).
2. Have the child repeat back what you said so you know they heard and encoded it.
3. Use written reminders — picture schedules, lists, checklists — and prompt the child to "check your list" rather than repeating yourself.
4. Rehearse what the child needs to remember just before the situation arises.
5. For middle schoolers, use cell phones, texts, or IMs as natural memory aids.
6. Help the child design their own memory system — what they think will work for them is more likely to be used.
7. Consider rewards for remembering and penalties for forgetting when working memory is only mildly weak.
8. When implementing a plan, establish the system with enthusiasm, cue generously, and fade supervision in the smallest possible increments.
9. Always follow up a verbal cue with a visual check to confirm the child has acted on it.
10. If you agree to let natural consequences happen (e.g., coach's reaction to missing equipment), do not rescue.
