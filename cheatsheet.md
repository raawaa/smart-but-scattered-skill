# Smart but Scattered — Cheatsheet

## ABC Intervention Angle

| If… | Then… | Why |
|---|---|---|
| Child **cannot** do it | **B — Teach** (Ch 7) | Skill deficit; motivation can't create missing skill |
| Child **can** but **won't** | **C — Motivate** (Ch 8) | Performance deficit; reward energizes practice |
| Environment is barrier | **A — Modify env** (Ch 6) | Least demand on child; external→internal |
| **Always start here** | **A first**, add B/C if needed | Change env before asking child to change |

## Skill Deficit vs. Performance Deficit

| Evidence | Diagnosis | Action |
|---|---|---|
| Never done it | Skill deficit | 6-step teaching sequence |
| Does it sometimes | Performance deficit | Incentives + environment |
| Used to, stopped | Performance deficit | Check incentives & consistency |
| Says "I can't" | Assume skill deficit | Investigate which skill is missing |

## 11 Executive Skills

| Skill | Type | Weakness sign |
|---|---|---|
| Response Inhibition | Doing | Blurts, acts without thinking |
| Working Memory | Thinking | Forgets instructions mid-task |
| Emotional Control | Doing | Meltdowns over small changes |
| Sustained Attention | Doing | Quits early, distracted |
| Task Initiation | Doing | Delays starting, needs prompting |
| Planning/Prioritization | Thinking | No attack plan |
| Organization | Thinking | Loses belongings, messy |
| Time Management | Thinking | Chronically late |
| Goal-Directed Persistence | Doing | Gives up at 1st obstacle |
| Flexibility | Doing | Rigid, can't adapt |
| Metacognition | Thinking | No self-awareness of own performance |

## Common Weakness Clusters

| Weaknesses | Entry point |
|---|---|
| Response Inhibition + Emotional Control | Teach stop-and-think first |
| Flexibility + Emotional Control | Preview transitions; build in choice |
| Initiation + Attention + Persistence | Shrink 1st step to 2 min; immediate reward |
| Time Mgmt + Planning | Visual schedules; backward-plan from deadline |
| Working Memory + Organization | Checklists; "a place for everything" |

## 10 Principles (Compact)

1. **Teach directly** — osmosis fails
2. **Consider developmental level**
3. **External → internal** — environment first
4. **Three levers**: env, task, interaction style
5. **Use mastery drive** — routines, choices, Grandma's Law
6. **Match effort** — design tasks to feel like 3 (1–10 scale)
7. **Incentives augment instruction**
8. **Just enough support** — child does most of the work
9. **Keep supports until mastery** (months, not weeks)
10. **Fade gradually** — hold, release seconds, catch wobbles

## Age-by-Age

| Age | Can do with support | Don't expect |
|---|---|---|
| 4–5 | Short bursts, basic self-care | Independent morning routine |
| 6–7 | 2-step instructions, homework with help | Self-initiation |
| 8–9 | Decreasing supervision, desk organize | Multi-step projects |
| 10–11 | Multi-step homework, projects w/ support | Independent time mgmt |
| 12–14 | Juggling subjects, long-term projects | Consistent independent perf. |

**Middle school**: Brain reorganizes 11–14. **Increase support, not decrease.**

## Which Routine First? (5 Criteria)

Pick the one that: (1) most improves quality of life, (2) small for quick success, (3) child chooses, (4) shareable with spouse, (5) aligns with long-term goals.

## 6-Step Teaching + 4-Stage Fading

**Steps**: (1) Define problem in observable terms → (2) Positive goal + baseline → (3) Outline steps → (4) Checklist/rule card → (5) Supervise + dry runs → (6) Fade support.

**Fading**: Parent directs → Parent points to checklist → Parent asks "What next?" → Child self-directs.

## Before-During-After

| Phase | Do |
|---|---|
| Before | Rehearse, prompt retrieval ("What's the rule?"), set visual cues |
| During | Coach with one word, cue card, monitor |
| After | Specific praise; debrief only when calm |

## Incentive Design (4 Steps)

1. Specific goal → 2. Reward menu (daily + weekly + long-term) → 3. Behavior contract → 4. Tinker

**3:1 positive:corrective ratio. "As soon as X, then Y."**

## Effort Scale (1–10)

| Score | Action |
|---|---|
| 3 = target | Child can do with effort |
| 5–7 | Break into smaller steps |
| 8–10 | Shrink task immediately |

## Parent-Child Fit

| You / Child | Risk | Opportunity |
|---|---|---|
| Strong / Weak | Irritation | Teaching advantage |
| Weak / Weak | Chaos | Empathy, mutual solving |
| Strong / Strong | Harmony | Praise openly |

**Stress rule**: Weakest skills fail first in both. Postpone new plans during illness, money, or marital stress.

## Diagnostic Audit (When Plan Fails)

1. Problem defined in observable what/when/where?
2. Realistic standard (small improvement OK)?
3. Age-appropriate for *this* child?
4. Visual cues + systems in place?
5. Specific skill identified?
6. Teaching procedure defined (who, how, how often)?
7. Incentives child actually values?

**14–21 day rule**: Stick with plan 2–3 weeks. Rate your consistency 1–5 daily.

## When to Seek Help

| Situation | Action |
|---|---|
| Sound plan failing >14–21 days | Behavioral/CBT clinician (not talk therapy) |
| School documents impairment | Testing for LD/ADHD |
| Friendships/self-esteem affected | Meds + behavioral combo |
| Need school accommodations | 504 Plan / IEP |

## Tells & Smells

- **"But you did it yesterday!"** → Inconsistency = skill weakness, not defiance
- **One-trial fallacy** → One explanation is never enough; this is normal
- **Extinction burst** → Behavior worsens before improving; ride it out 14–21 days
- **Over-debriefing** → Heightens anxiety; debrief sparingly, at distance
- **Words alone fail** → Always pair verbal with visual cues (list, schedule, picture)
- **"It's easy"** → Invalidates child's experience; assume they're truthful
